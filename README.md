# ✨ Subdomain Automation ✨

> **Automate with ❤️ and ✨**

A clean, aesthetic automation wrapper for `subfinder` and `subdominator`.

##  Usage

```bash
# Single Domain
bash subfinder+subdominator.sh -d example.com

# List of Domains
bash subfinder+subdominator.sh -dL domains.txt

# Save to File (Optional)
bash subfinder+subdominator.sh -d example.com -o results.txt
```
